    webpack核心

    1. 入口：指示webpack应该使用哪个模块，来作为构建其内部依赖图的开始。

    2. 输入：指示webpack在哪里输出它所创建的 bundles。

    3. loader：loader 让 webpack 能够去处理那些非 JavaScript 文件。

    4. plugin：webpack功能的扩展，可以用来处理各种各样的任务。
