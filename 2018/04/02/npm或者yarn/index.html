    npm是什么？npm能做什么？

    npm（node package manager），通常称为node包管理器。顾名思义，它的主要功能就是管理node包，包括：安装、卸载、更新、查看、搜索、发布等。

    那么yarn又是什么呢？

    yarn是一个新出现的包管理工具，解决了了npm的历史遗留问题，具有更高效快速的特点。

    怎么使用它们

    npm安装直接安装node.js就行了，它是与node捆绑在一起的。
