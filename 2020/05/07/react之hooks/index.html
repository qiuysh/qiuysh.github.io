    • react之hooks

    今年以来前端团队推荐使用hooks，恰逢新开发一个产品，使用了hooks体验了一番。
    首先要说的hooks仅限用于函数组件，那么下面就说说使用频繁的几个。

    一、useState

    useState的出现解决了函数组件的state使用问题。

    二、useEffect

    一般会被认为是componentDidMount、componentDidUpdate和componentwillUnmount的集合体。
    这种说法也不算错。

    三、useRef

    如果你不想粗暴的去获取dom，而是更为优雅的使用dom，你可以试试useRef，当然如果是class组件可以用createRef

    四、useReducer

    五、useImperativeHandle/forwardRef

    forwardRef可以暴露整个子组件给父组件

    配合useImperativeHandle成对使用的情况下，可以暴露指定的属性或方法给父组件

    六、memo

    memo函数式组件的优化方法，类似class组件中PureComponent

    更多的hooks就不一一介绍了。

    https://react.docschina.org/docs/hooks-intro.html
